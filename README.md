# vimrc

My Vim configuration

## Quickstart

```bash
# backup old configuration (optional)
mv ~/.vimrc ~/.vimrc.bkp

# clone repository
cd ~/Documents
git clone https://github.com/sskender/vimrc.git

# link vimrc file
ln -s ~/Documents/vimrc/vimrc ~/.vimrc

# update
cd ~/Documents/vimrc
git pull
```

### Docker

```bash
docker pull sskender/vimrc
docker run -it sskender/vimrc
```
